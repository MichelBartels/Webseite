PHP


Was ist PHP:
PHP(Hypertext Preprocessor, ursprünglich Personal Home Page Tools) ist eine Skriptsprache, die hauptsächlich zur Erstellung dynamischer Webseiten verwendet wird. Dies bedeutet, das eine Kommunikation mit dem Server möglich ist, z.B. in Form eines Chats. Auch gibt es Funktionen die dem Besitzer der Website erlauben, z.B. die Anzahl von Besuchern zu zählen.

Wie Funktioniert es:
Um PHP anwenden zu können, muss der PHP-code in einer PHP Datei geschrieben werden. Diese PHP-Datei funktioniert wie eine Normale HTML-Datei, nur dass auch PHP-Code verwendet werden kann. Der PHP-Code wird dann in einen solchen tag geschrieben: .
Damit dies funktioniert muss die Datei jedoch auf einem Server, der PHP unterstützt, liegen. Zudem sind weitere Kleinigkeiten zu beachten, wie z.B, dass eine Variable immer mit einem $-Zeichen beginnt und, dass das Gleichheitszeichen nicht im mathematischen Sinne zu verstehen ist, sondern als Operator für Zuweisungen gedacht ist.
